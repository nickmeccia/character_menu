    DESCRIPTION: Create a series of webpages with the attached. Each section of text should be a separate page. Create an index page to link to each.     Layout

    4 points | due Mon, Feb 11

    OBJECTIVE: Create hierarchy through layout and beautiful typography.

    DESCRIPTION: Create websites using the attached content. Different sections of content are clearly labeled as to their function, but not for their importance.     • TURN IN:

    • * 1 web page for each site, based on the approved sketch
